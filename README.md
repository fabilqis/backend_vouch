# backend_vouch

| Endpoint           | Http          | Description  |
| -------------      |:-------------:| ------------:|
| /                  | GET           | Home         |
| /tickets           | GET           | Ticket Info  |
| /tickets/:stats    | GET           | Check Status |
| /tickets/add       | POST          | Add Ticket   |
| /tickets/delete    | POST          | Delete Ticket|
| /tickets/update    | POST          | Update Logs  |

## Stacks
- MongoDB
- Express.js
- Nodemon
- Mongoose
- cors
